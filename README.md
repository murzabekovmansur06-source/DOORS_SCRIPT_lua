## ✨ Возможности

### 🏃 Движение
- **Плавный WalkSpeed (17-21)** — скорость меняется плавно через TweenService, без палева
- **Луч-Телепорт (NoClip)** — нажми кнопку и телепортируйся туда, куда смотрит камера
- **Телепорт к выходу** — мгновенное перемещение к ближайшей двери

### 👁️ ESP (Подсветка)
- **Сущности (Красные)** — Rush, Ambush, Figure, Seek, Screech, Halt, Eyes, Timothy, Jack
- **Предметы (Зелёные)** — ключи, монеты, батарейки, отмычки, витамины, распятия, свечи
- **Двери (Жёлтые)** — все двери на карте
- **Игроки** — подсветка других игроков

### 🔍 Тройной детектор монстров
ESP использует три независимых метода обнаружения для максимальной точности:
1. **По имени** — ищет ключевые слова в названии объекта (Rush, Ambush, Figure, Seek, Screech)
2. **По звуку** — анализирует звуковые компоненты монстров
3. **По атрибутам** — проверяет специальные атрибуты объектов (IsRush, IsAmbush и др.)

Разные цвета для разных монстров:
- Rush — оранжевый 🔶
- Ambush — розовый 🩷
- Figure — тёмно-красный ♦️
- Seek — ярко-красный 🔴
- Screech — фиолетовый 🟣

### 🎨 Визуал
- **FullBright** — максимальная яркость, никакой темноты
- **NoFog** — убирает туман для лучшей видимости

### 🤖 Автоматизация
- **Авто-Взаимодействие** — автоматически нажимает ProximityPrompt (кнопки, предметы, исключая шкафы и кровати)
- **Анти-Screech** — автоматически поворачивает камеру к Screech
- **Анти-AFK** — бот не даст игре кикнуть тебя за бездействие

### 🛡️ Защита
- **God Mode** — бессмертие (Health = MaxHealth каждые 0.1 сек)
- **Анти-Screech** — авто-поворот к Screech при его появлении

## 🚀 Установка

1. **Скопируй** код из файла `Doors_Hub_v2.lua`
2. **Вставь** в свой экзекутор
3. **Запусти** игру DOORS
4. **Нажми** кнопку 🚪 слева на экране для открытия меню

**Готово! Хаб активирован.**

---

## 🎮 Как пользоваться

1. После запуска скрипта нажми на фиолетовую кнопку 🚪 в левой части экрана
2. Откроется красивое меню с категориями
3. Включай нужные функции тумблерами (по умолчанию всё выключено)
4. Для телепорта используй кнопки действий
5. Чтобы скрыть меню — нажми ✕ в заголовке

---

## 📱 Оптимизировано для мобильных

- Кнопка вызова меню расположена удобно для большого пальца
- Меню 300x420 пикселей — не перекрывает весь экран
- Плавные анимации переключателей
- Тени и скругления как в премиум-приложениях
- ESP сканирует только объекты в радиусе 60 studs для производительности

## ⚙️ Технические детали

| Функция | Метод | Безопасность |
|---------|-------|-------------|
| WalkSpeed | TweenService (плавное изменение) | ✅ Низкий риск |
| Телепорт | Raycast + CFrame | ⚠️ Средний риск |
| ESP | Highlight (стандартный объект Roblox) | ✅ Безопасно |
| GodMode | Health = MaxHealth | ⚠️ Средний риск |
| FullBright | Lighting.Brightness | ✅ Безопасно |
| Детектор монстров | 3 метода (имя/звук/атрибуты) | ✅ Безопасно |

---

## ⚠️ Предупреждения

**Следующие функции могут работать нестабильно:**
- **Тройной детектор монстров** — не тестировался на всех версиях DOORS, возможны ложные срабатывания
- **God Mode** — некоторые версии игры могут детектить частое обновление здоровья
- **NoClip (Луч-Телепорт)** — работает только при активации кнопкой, не пассивный режим

Если какая-то функция не работает — создайте Issue на GitHub или напишите в Discord.

---

## ⚠️ Важно

- Скрипт работает **только на клиенте** (LocalScript)
- Не требует серверных привилегий
- Совместим с FilteringEnabled (FE)
- Используйте с осторожностью на публичных серверах
- Все функции по умолчанию **выключены** — включайте только нужное
